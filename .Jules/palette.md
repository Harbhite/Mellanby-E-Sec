\n## 2026-03-17 - Enhance Mobile Menu Button Accessibility\n**Learning:** When using Lucide React icons within button controls, they should be explicitly hidden from screen readers using `aria-hidden="true"` to prevent redundant/confusing announcements, and the button itself needs a dynamic `aria-label` alongside `aria-expanded` and `aria-controls`.\n**Action:** Ensure all future toggle buttons (especially those containing icon-only content) have explicitly hidden SVGs and clearly defined semantic relationships via ARIA attributes.
